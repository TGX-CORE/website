# has

Checks whether the array has the matched value or passes the function.

## Constructor
 ```ts
 has( array, value )
 ```
 
 | Parameter | Type | Description |
| :--- | :--- | :--- |
| `array` = `[]` | `any[]` | The array that contains the value. |
| `value` | `any` | The value to check or the validator. |

Return: boolean
